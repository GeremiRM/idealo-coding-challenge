Toy Robot Challenge

Rules of the game

1. To control the robot you can only use the following commands:
POSITION [x] [y] [dir], FORWARD [1|2|3], TURNAROUND, WAIT, RIGHT*

2. If your instructions lead to the robot going out of the limits of the grid, a new robot will appear back at the beginning. What happens to the robots after they fall off the grid remains a mistery.

* If you're not in a yelling mood, you can write the instructions in lowercase as well.
